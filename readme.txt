Guest Post Plugin will help you to generate a guest posting form in the simple and easiest way.

== Description ==
Use Guest Post Plugin and allow your visitors to submit posts without registration . Anyone will be able to submit post and it will be added automatically as pending post. 
After reviewing you will be decide which posts approved for publish and which are rejected. Submitted posts will be included with name, email address, post title and other infos and the for the published post the
the guest will get an email after it is published and alongwith that he/she will also get an pdf attached with his/her submitted information. 


<h3>All Features:</h3>
<ul>
	<li>Use this shortcode [guest-post] to display the form and accept submissions from frontend.</li>
	<li><strong>Guest Post Plugin</strong> will create a form where your guests can submit their posts.</li>
	<li>Posts will be directly saved into WordPress database and will show up in Admin Dashboard as pending post with given <strong>Title</strong>.</li>
	<li>Moderator will be able to review and approve them as needed.</li>
	<li>Name and email will be added as custom field.</li>
	<li>After post is published an email notification will be sent to the guest with the submitted data in a pdf alongwith an email template.</li>
	<li>Simple and user friendly</li>
</ul>


<strong>Note: </strong> Please read full description for better use.

<h3>More User Guide</h3>

Create a new page/post with the name "Simple Guest Post" or whatever you like best.

If you don’t want to use custom redirect url or author then use this shortcode  <code>[guest-post]</code> into the page/post and publish it. 
It will publish post under admin and redirect will be to homepage.

== Installation ==

You will be able to add so simple and user friendly guest posting feature for your visitors within a short moment.

1. From your WordPress Dashboard Go for a "Add New" inside "Plugins" menu.

2. Search for "Guest Post Plugin" and install the plugin.

3. Activate the plugin

<strong>Or</strong>

1. Upload `guest-post.zip` to the `/wp-content/plugins/` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.

Nice!!! Plugin upload and activation have been completed!!!


== Frequently Asked Questions ==

= why my shortcode is not working? =

The shortcode is not working because you may not have created a user with contributor role.

= I have created a user still my shortcode is not working =

Its because you have not selected a user for as default author for post to save.
You can assign a user under guest post->settings.
